Este é um projeto Python que permite analisar o uso de espaço em disco em um diretório específico, 
identificando arquivos grandes e fornecendo informações detalhadas sobre o tamanho total de arquivos e pastas.
O objetivo do projeto é ajudar os usuários a gerenciar melhor o espaço em disco,
identificando arquivos que estão ocupando muito espaço e oferecendo uma visão geral do uso do sistema de arquivos.

- Análise de Tamanho: Verifique o uso de espaço de arquivos e pastas em um diretório especificado.
- Identificação de Arquivos Grandes: Encontre arquivos que ocupam mais espaço, útil para liberar espaço em discos.
- Interface Gráfica: Use uma interface gráfica simples para facilitar o uso, com botões e menus intuitivos.

Essas instruções irão ajudá-lo a configurar e executar o projeto em sua máquina local para desenvolvimento ou testes.

1. Clone este repositório:

    ```bash
    git clone https://github.com/DomColabone/analisador-espaco.git
    ```

2. Acesse a pasta do projeto:

    ```bash
    cd analisador-espaco
    ```

3. Instale as dependências listadas no `requirements.txt`:

    ```bash
    pip install -r requirements.txt
    ```

4. Agora, você pode executar o script Python:

    ```bash
    python analisador_espaco.py
    ```

1. O programa solicitará que você digite o caminho do diretório que deseja analisar.
2. O programa exibirá informações sobre o uso total de espaço e uma lista dos arquivos grandes encontrados.
3. Utilize a interface gráfica para navegar e interagir facilmente com o programa.
